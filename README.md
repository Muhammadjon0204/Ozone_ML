## 📊 Описание исследования
В рамках данного проекта разработана модель машинного обучения для предсказания концентрации озона в атмосфере. В качестве основы взят массив данных за 2024 год от агентства **EPA (США)**.

* **Объем данных:** более 380,000 строк.
* **Задача:** Регрессия (предсказание числового значения концентрации озона).

## 🛠 Технологический стек
* **Язык:** Python 3.14
* **Библиотеки:** `pandas`, `scikit-learn`, `matplotlib`
* **IDE:** JetBrains PyCharm 2026.1

## 🚀 Этапы реализации
1.  **Предобработка данных:** Выбор значимых признаков (`Latitude`, `Longitude`, `AQI`, `Observation Count`, `Observation Percent`).
2.  **Обработка пропусков:** Использование `SimpleImputer` (стратегия `mean`) для заполнения пустых ячеек средними значениями, что сохранило целостность выборки.
3.  **Модель:** Алгоритм **RandomForestRegressor** (Случайный лес). Выбрано 50 деревьев решений для баланса точности и скорости.
4.  **Оптимизация:** Применено параллельное вычисление (`n_jobs=-1`), что позволило эффективно использовать ресурсы процессора при обучении на 300к+ строках.
5.  **Визуализация:** Создан интерактивный отчет в формате `.ipynb` с графическим анализом результатов.

## 📈 Результаты
Модель показала высокую точность предсказания:
* **Средняя абсолютная ошибка (MAE):** `0.003309`

Это означает, что предсказания модели отклоняются от реальных данных мониторинга в среднем лишь на **0.0033**, что является отличным показателем для экологических прогнозов.
