# Исследование качества воздуха (Озон 2024 год)

Проект выполнен студентом "Косимов Мухаммад" в рамках домашнего задания по предмету "Системы ИИ".

## Описание исследования
В качестве набора данных использованы ежедневные сводки мониторинга озона в США за 2024 год (EPA Daily Summary Data).
Объем данных: более 380,000 строк.

## Этапы работы:
1. **Предобработка данных:** Загрузка и очистка признаков.
2. **Обработка пропусков:** Использован `SimpleImputer` со стратегией `mean` для заполнения отсутствующих значений в данных мониторинга.
3. **Модель:** Алгоритм `RandomForestRegressor` (Случайный лес) с 50 деревьями решений.
4. **Оптимизация:** Применено параллельное вычисление (`n_jobs=-1`) для ускорения обучения на большом массиве данных.

## Результаты
Модель показала среднюю абсолютную ошибку (**MAE**): **0.003309**.
Это свидетельствует о высокой точности предсказания уровня концентрации озона на основе географических координат и индекса AQI.
